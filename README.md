# Mall Customer Spending Analysis in R

## Project Overview

This project analyzes customer demographics and spending behavior using R. The aim is to identify factors that influence customer spending through descriptive statistics, data visualization, correlation analysis, and hypothesis testing.

## Dataset

The dataset contains information on 200 mall customers, including:

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

## Objectives

- Explore customer demographics.
- Analyze the distribution of numerical variables.
- Examine relationships between age, annual income, and spending score.
- Compare spending behavior between male and female customers.
- Perform statistical tests to support findings.

## Methods Used

- Descriptive Statistics
- Histograms
- Boxplots
- Scatterplots
- Correlation Analysis
- Independent Samples t-test
- Data Manipulation using dplyr

## Tools Used

- R
- RStudio
- tidyverse
- ggplot2
- dplyr

## Key Findings

- Age and annual income have a weak negative relationship.
- Annual income and spending score have a weak positive relationship.
- No statistically significant difference was found in annual income between male and female customers.
- No statistically significant difference was found in spending scores between male and female customers.
